CRTL + D : 원한는 단어에서 선택할경우 같은 단어들을 한개씩 늘려가며 수정 가능하게 처리 해줌
CTRL + SHIFT + L : 원하는 단어에서 같은 단어들을 한번에 선택하게 처리 해줌

ALT + CLICK : 원하는 곳을 클릭하면 커서를 생성 시켜 줘서 한꺼번에 처리 하게 해줌
ALT + ARROW : 해당 라인을 위아래로 이동 시켜줌
ALT + SHIFT + ARROW : 해당 라인을 위아래로 복사 시켜줌
ALT + SHIFT + I : 블럭된 라인에 커서를 생성해서 한꺼번에 처리 하게 해줌
ALT + SHIFT + DRAG : 클릭하는 영역까지 커서를 생성해줌

CTRL + [] : 들여쓰기, 내어쓰기
CTRL + / : 선택된 라인 주석처리 하기

CTRL + SHIFT + K : 해당 라인 삭제하기
